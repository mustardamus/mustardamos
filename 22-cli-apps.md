# CLI Apps

## Htop

https://htop.dev/

```shell:terminal
sudo apt update
sudo apt install htop -y
```

## `lazydocker`

https://github.com/jesseduffield/lazydocker

```shell:terminal
asdf plugin add lazydocker https://github.com/comdotlinux/asdf-lazydocker.git
asdf install lazydocker latest
asdf global lazydocker latest
```
