# LightDM

## Installation

```shell:terminal
sudo apt update
sudo apt install lightdm -y
```

- Reboot the machine
- Login as non-root user
- Select `Regolith/X11` as session in the top right corner
- Bring up the terminal with `Win-Return`
- And continue with notebook `./12-lightdm-setup.md`

```shell:terminal
sudo reboot now
```
